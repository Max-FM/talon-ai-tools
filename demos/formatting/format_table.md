# Table Formatting Demo

Open this file in a text editor, highlight the text to be formatted, and use the below Talon command to see the formatting in action.

## Command to Use

`model format table`


## Text to Format

Apple 1.20 10
Banana 0.50 25
Orange 0.80 15
Grapes 2.50 8
Pineapple 3.00 5

## Example Output

| Fruit     | Price | Quantity |
| --------- | ----- | -------- |
| Apple     | 1.20  | 10       |
| Banana    | 0.50  | 25       |
| Orange    | 0.80  | 15       |
| Grapes    | 2.50  | 8        |
| Pineapple | 3.00  | 5        |
